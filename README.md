# AI Village Risk Detection & Resource Allocation

## Overview
An AI-driven framework for rural risk prediction and intelligent resource allocation using the Mission Antyodaya Survey dataset. The system identifies high-risk villages and prioritizes resource distribution using machine learning.

## Problem
Government resource allocation in rural areas is often not data-driven, leading to inefficient investment and unequal distribution of funds.

## Solution
This project:
- Predicts village-level risk scores
- Prioritizes villages for resource allocation
- Applies fairness constraints
- Incorporates budget limitations
- Provides visualization for decision-making
